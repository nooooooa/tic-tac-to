# tic-tac-to

A console-based Tic-Tac-Toe game developed in C. This project allows players to compete against an AI opponent with two difficulty levels:

Easy Mode: The AI makes random moves.
Hard Mode: The AI uses a smarter strategy to provide a more challenging gameplay experience.


Features

+ Single-player Tic-Tac-Toe game
+ Two AI difficulty levels (Easy and Hard)
+ Win, lose, and draw detection
+ Simple command-line interface
+ Input validation to prevent invalid moves


Technologies Used 
+ Programming Language
+ Standard C Libraries

---

How to Run

1. Compile the source code:

```gcc main.c -o tictactoe```

2. Run the program:

```./tictactoe```

Learning Objectives

This project was created to practice:

+ Arrays and matrix manipulation
+ Functions and modular programming
+ Conditional statements and loops
+ Basic artificial intelligence implementation in C
+ User input validation
